# 📱 Textie

Textie is a high-performance, cross-platform mobile messaging application engineered for seamless, instant communication. Built with **React Native**, **TypeScript**, and **NativeWind**, it delivers a fluid, native-feeling user experience across both iOS and Android devices.

At its core, Textie leverages a robust **WebSocket architecture** to handle real-time message delivery, live synchronization, and instant user-presence lifecycles with minimal latency.

---

##  Features

* **Real-Time Messaging:** Instant message delivery and receipt handling powered by WebSockets.
* **User Presence Lifecycles:** Live updates showing who is online, offline, or typing.
* **Cross-Platform Performance:** Shared codebase delivering native iOS and Android performance via React Native.
* **Type-Safe Architecture:** Built from the ground up with TypeScript for reliable, predictable code.
* **Modern Styling:** Clean, adaptive UI built using NativeWind (Tailwind CSS for React Native).
* **Offline Synchronization:** Graceful handling of network drops with message synchronization upon reconnection.
